# EV Digital Twin

This repository contains files related to the EV Digital Twin project, including prediction models, IoT integration, and web interface.

---

## 📁 Project Files

### 🔧 Python Scripts
- `EV_PredictionFunction.py`  
  Handles EV prediction logic (e.g., battery, performance, or data analysis).

- `publish_to_iot2.py`  
  Script for publishing data to IoT platform.

---

### 📊 Configuration Files
- `IoTTwinmaker.json`  
  Configuration for AWS IoT TwinMaker or digital twin setup.

- `random_forest_params (1).json`  
  Stores parameters for the Random Forest model.

---

### 🌐 Web Interface
- `index.html`  
  Main frontend page for visualization or interaction.
